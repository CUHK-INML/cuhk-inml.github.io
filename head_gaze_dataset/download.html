Download the dataset

Before the downloading, please make sure you have read and agree with the Terms of Use of the Head and Gaze VR Behavior Database.

If you're interested in downloading the full set of Videos Sequences (with music), please download here.

If you only need to download the subset of head and gaze data that we tracked, we provide a Python script that downloads the tracking data from website.
  1. Download the file downloader.py (press Ctrl + S), or directly run: 
    wget https://raw.githubusercontent.com/google/aistplusplus_api/main/downloader.py
  2. Run the following script: 
    python downloader.py --download_folder=$DOWNLOAD_FOLDER --num_processes=5
    For help, run: 
    python downloader.py -h
